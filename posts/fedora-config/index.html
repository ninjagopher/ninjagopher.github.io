Fedora Workstation 41 安装安装之后的配置

配置 DNF#

sudo vi /etc/dnf/dnf.conf # 编辑 dnf.conf
# 添加下面的内容
[main]
fastestmirror=True
max_parallel_downloads=16
defaultyes=True # 默认yes，用户按下 Enter 键继续
keepcache=True
# assumeyes=True 无交互，直接yes

~/.bashrc 配置#

sudo vi ~/.bashrc                      # 编辑 ~/.bashrc
# 、、
bind "set completion-ignore-case on"   # Tab 补全忽略大小写
export HISTCONTROL=erasedups           # 历史命令不重复
export HISTIGNORE="&:ls:cd:pwd:clear"  # 忽略不重要的命令
source ~/.bashrc                       # 应用修改

配置 Grub 启动项（Dual Boot）#

sudo vi /etc/default/grub                 # 编辑 /etc/default/grub
# 根据需要修改下面两行
GRUB_TIMEOUT=30
GRUB_DEFAULT=saved
# 添加下面这行
GRUB_SAVEDEFAULT=true
sudo -i                                   # 切换到 oot 用户
grub2-mkconfig -o /boot/grub2/grub.cfg    # 生成新的 GRUB 配置文件

更新#

dnf repolist                # 查看软件仓库
sudo dnf check-update       # 检查更新
sudo dnf update             # 更新

DNF 的使用#

# 查找某个软件包
dnf repoquery <pkg-name>
dnf info <pkg-name>
dnf list --installed <pkg-name> # 查找已安装的软件包
dnf remove <pkg-name>           # 卸载
dnf autoremove                  # 清理无用的依赖
dnf repolist                    # 查看启用的软件仓库
dnf --help                      # 查看帮助

启用 RPM Fusion & 安装 VLC#

参考 RPM Fusion Configuration

  1. 添加 rpm fusion 软件仓库（包含自由软件和非自由软件）
sudo dnf install https://mirrors.rpmfusion.org/free/fedora/rpmfusion-free-release-$(rpm -E %fedora).noarch.rpm https://mirrors.rpmfusion.org/nonfree/fedora/rpmfusion-nonfree-release-$(rpm -E %fedora).noarch.rpm
  1. 启用 Cisco 的 OpenH264 编解码器
sudo dnf config-manager setopt fedora-cisco-openh264.enabled=1
  1. 更新 “核心系统” 相关的软件包
sudo dnf update @core   # 更新 @core 软件包组
dnf group info core     # 查看这个软件包组有哪些包
  1. 安装 VLC
sudo dnf install vlc

多媒体相关#

参考 RPM Fusion Multimedia

  1. 将 ffmpeg-free 替换为 ffmpeg，并允许删除可能存在的软件包冲突
sudo dnf swap ffmpeg-free ffmpeg --allowerasing
  1. 更新 Fedora 系统中的 @multimedia 组的软件包
sudo dnf update @multimedia --setopt="install_weak_deps=False" --exclude=PackageKit-gstreamer-plugin
  1. 替换 Mesa 驱动程序，以获得更广泛的视频解码支持（AMD 硬件加速）
sudo dnf swap mesa-va-drivers mesa-va-drivers-freeworld
sudo dnf swap mesa-vdpau-drivers mesa-vdpau-drivers-freeworld
# 通过替换 Mesa 的 VA-API 和 VDPAU 驱动程序，从 Fedora 默认仓库切换到 RPM Fusion 的 freeworld 仓库，从而获得更全面的硬件加速视频解码支持。

调节外接显示器亮度#

dnf search ddcutil
sudo dnf install ddcutil        # 安装 ddcutil
sudo ddcutil detect
sudo ddcutil setvcp 10 <value>  # 调节亮度，<value> 通常在 0 到 100 之间，表示亮度的百分比。
flatpak install flathub com.mattjakeman.ExtensionManager   # 安装扩展管理器
# 最后安装 Brightness control using ddcutil 扩展

安装字体#

在 Nerd Fonts 下载 JetBrainsMono Nerd Font 字体压缩包

sudo mkdir -p /usr/share/fonts/jetbrains-nerd-fonts     # 创建字体文件夹
sudo unzip ./JetBrainsMono.zip                          # 解压
sudo mv ./*.ttf  /usr/share/fonts/jetbrains-nerd-fonts  # 移动.ttf文件到对应的目录
fc-cache -fv                                            # 更新字体缓存
fc-list | grep JetBrains                                # 检查是否成功加载
